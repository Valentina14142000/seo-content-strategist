SEO Content Strategist Agent
An autonomous multi-agent system built with LangGraph designed to automate end-to-end SEO content strategy. This tool streamlines the workflow from initial keyword research to generating high-performing, structured content briefs.

Features
Autonomous Research: Uses Tavily search to scrape competitor data and identify keyword gaps.

Strategic Planning: Automatically generates structured content briefs including H1/H2 hierarchies and target audience definitions.

Agentic Workflow: Implements a Supervisor/Worker pattern for modular, scalable, and debuggable AI operations.

Extensible Architecture: Easily add new nodes for internal linking, social media repurposing, or automated publishing.

Tech Stack
Framework: LangGraph (State management & workflow orchestration)

Language: Python 3.13

LLM Integration: LangChain / OpenAI

Search Engine: Tavily API
